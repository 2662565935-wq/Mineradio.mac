# Mineradio.mac

本项目是为 Mac 用户定制的 `Mineradio` 适配与移植版本。

### 📢 声明 (Statement)
本项目的核心功能、品牌及代码版权归原作者所有。由于原项目目前未提供官方的 macOS 版本，本项目旨在为 Mac 用户（特别是 **Apple Silicon / M系列芯片**）提供开箱即用的 `.dmg` 编译版本与运行环境适配。

- **官方原项目**:(https://github.com/XxHuberrr/Mineradio)

---

### 📦 下载与安装 (Download)
请点击页面右侧的 **[Releases]** 区域，下载最新的 `Mineradio-1.1.1-arm64.dmg` 安装包。
